# Data Salaries Dashboard

An interactive **Streamlit** dashboard to explore salary trends in data-related roles. Users can filter by **year, seniority, contract type, and company size**, and visualize key metrics and charts with Plotly. Can be accessed on https://data-salary-analysis-dashboard-jqkiasd9sagjgb2ff24caz.streamlit.app/

---

## Features

- Filter salaries by **year**, **seniority**, **contract type**, and **company size**  
- Display **key metrics**: average salary, maximum salary, total records, most common role  
- Interactive **charts**:  
  - Top 10 roles by average salary  
  - Salary distribution histogram  
  - Proportion of remote vs on-site roles  
  - Average Data Scientist salary by country (choropleth map)  
- Detailed **data table** for in-depth exploration  

---

## Libraries Used

- Streamlit– for building the interactive web app  
- Pandas– for data manipulation  
- Plotly– for interactive charts and visualizations
